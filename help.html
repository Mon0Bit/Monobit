GETTING STARTED

Welcome to MonoBit Arcade! This minimal retro gaming hub features classic games in pure black and white. Navigate through our collection and start playing instantly.

HOW TO PLAY

  • Click on any game card to view details
  • Press the PLAY button to launch the game
  • Each game opens in a new window
  • Use keyboard or mouse controls as indicated

THEME CONTROLS

Customize your viewing experience with our retro display modes:

CRT MODE Adds scanlines for authentic CRT monitor effect
INVERT Swaps black and white colors (white background)
B&W Pure black & white high contrast mode
SFX Toggle sound effects on/off
MUSIC Toggle background music on/off

Combine modes for different effects! Try CRT + INVERT for a classic green screen terminal look.

NAVIGATION

KEYBOARD SHORTCUTS

  • ARROW KEYS: Navigate between game cards
  • ENTER/SPACE: Select focused game
  • TAB: Move through interactive elements
  • ESC: Close game windows (browser dependent)

MOUSE CONTROLS

  • CLICK: Select games and buttons
  • HOVER: Preview game cards with sound effect
  • SCROLL: Navigate through game library

TOUCH CONTROLS (MOBILE)

  • TAP: Select games and controls
  • SWIPE: Scroll through content
  • Haptic feedback on supported devices

FREQUENTLY ASKED QUESTIONS

Why is the sound not working?
Browsers require user interaction before playing audio. Click anywhere on the page first, then toggle the SFX or MUSIC buttons. Make sure your device is not muted.
Can I play on mobile devices?
Yes! All games are optimized for mobile play. Some games may work better in landscape mode. Touch controls are supported throughout.
Do games save progress?
Individual games may have their own save features. Check each game's instructions. Browser storage is used when available.
Why black and white only?
MonoBit embraces minimal retro aesthetics. The monochrome design reduces visual noise and creates a focused, nostalgic gaming experience.
Are there more games coming?
Yes! We're working toward our goal of 100 games. New retro classics are added regularly. Check back often for updates.
How do I report bugs or issues?
Visit our GitHub repository (link in footer) to report issues or contribute. Your feedback helps improve the arcade experience.

TIPS & TRICKS

  • Use CRT mode for reduced eye strain during long play sessions
  • Turn off sound effects in quiet environments
  • Inverted mode can be easier on the eyes in dark rooms
  • Most games support both keyboard and mouse controls
  • Bookmark your favorite games for quick access
  • Clear browser cache if games don't load properly
  • Try different theme combinations for varied experiences
  • Use keyboard navigation for faster game selection

ACCESSIBILITY

We strive to make MonoBit accessible to all players:

  • Full keyboard navigation support
  • Screen reader compatible markup
  • High contrast black/white design
  • Adjustable visual modes (CRT, Inverted, Pure B&W)
  • Optional sound effects and music
  • Responsive design for all screen sizes
  • Touch and mouse support
